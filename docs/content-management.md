# RobAI-Club 内容管理平台

管理入口：`https://logancome.github.io/robailab.github.io/admin/`

管理界面固定使用 Decap CMS `3.14.1`。暂不升级到 `3.15.0`，该版本存在保存后界面崩溃的已知回归。

## 权限与工作流

- 登录使用成员自己的 GitHub 账号，不共享网站密码。
- GitHub 仓库权限是唯一授权来源；移除协作者后，其后台写入权限同步失效。
- 新建或编辑内容先保存为草稿，再进入审核，最后由有权限的成员发布。
- 每次修改都保留 Git 提交和审核记录，可以追溯和回滚。

## 当前部署说明

后台“发布”会把审核通过的内容合并到 `main`。由于该 GitHub 账号目前无法运行 Actions，线上 `gh-pages` 仍需由维护者执行一次本地 Hugo 构建与部署；账号的 Actions 恢复后，这一步可重新自动化。此限制不影响编辑、上传、审核和版本记录。

## 可管理内容

- 中英文固定页面；
- 中英文协会动态；
- 成员资料与头像；
- 论文、开源项目和技术成果；
- 竞赛荣誉与证书图片；
- 公开文件库及页面正文中的附件。

## 上传规则

- 头像：建议使用清晰的正方形或竖向照片；
- 获奖证书：建议使用 `3508 × 2480` 或相同比例的横向图片；
- 新闻图片：上传前应压缩，不建议超过 3 MB；
- 上传到本后台的所有文件都会进入公开 GitHub 仓库，不得上传身份证件、账号密码、内部名单或其他敏感材料。

## 双语发布建议

同一条新闻应分别创建中文和英文版本，并使用相同的 URL 标识。没有核实的英文翻译可以先保留为草稿，不应为了凑齐双语内容而编造信息。
